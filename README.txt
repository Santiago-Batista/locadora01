## Breve explicação do sistema
Sistema desenvolvido para uma locadora de carros.

O sistema permite:

- realizar login com usuários cadastrados;
- cadastrar carros;
- listar carros;
- editar carros disponíveis;
- excluir carros disponíveis;
- registrar agendamento de locação;
- finalizar agendamento;
- atualizar automaticamente o status do carro.

Quando o carro está disponível, pode ser agendado, editado ou excluído.

Quando o carro está agendado, pode apenas finalizar o agendamento, retornando para disponível.
